**Subject:** The 3 fields that price a Japanese card
**Preview text:** Why artwork-first ID costs you money

<p>Two Japanese Charizard cards. Identical art. One is $40, one is $420. The difference lives in two fields at the bottom of the card — and if you're reading artwork instead of collector numbers, you'll miss it every time.</p>

<p>I just published a full anatomy guide on Japanese Pokémon cards — what each field means, where it sits, and exactly which ones matter for pricing. This is the condensed version. The directive version.</p>

<h2>The only three fields that matter for identification</h2>

<p><strong>Card name (top-left).</strong> Written in Katakana. You don't need to read it — it maps directly to the English equivalent in any database. What matters is that it sits in the same position as English cards, so a scanner knows where to look.</p>

<p><strong>Collector number (bottom-right).</strong> Formatted as <em>XXX/YYY</em> — your card's position in the set over the base set total. If that first number exceeds the second — say, <em>072/071</em> — you're holding a secret rare by definition. That single check separates a $0.25 common from an $80–$350+ pull before you do anything else.</p>

<p><strong>Set code (bottom-left).</strong> A short alphanumeric string near the legal line. <em>SV8a</em>. <em>SV9</em>. Combined with the collector number, this is the unique identifier for every print run. It also tells you whether an English equivalent exists yet — Japanese sets frequently ship 2–6 months before their English counterparts, which means TCGplayer data may be sparse or absent entirely.</p>

<p>That's the full identification framework. Name + set code + collector number. Everything else on the card is gameplay data.</p>

<h2>Where the common workflow breaks down</h2>

<p>The standard advice floating around is to use Google Translate. For flavor text, fine. For pricing, it fails in a specific way: set codes and collector numbers are alphanumeric strings, not natural language. Translate frequently skips or misreads them — the exact fields you need most.</p>

<p>The other failure mode is rarity symbol identification without checking the collector number first. Two cards can share a ★★★ symbol but carry different numbers that put one at $30 and the other at $200+. The symbol is a tier signal. The number is the identifier. Check number first, symbol second.</p>

<h2>One more field most guides bury: the regulation mark</h2>

<p>Since 2021, Japanese cards have included a single letter — D, E, F, G, H — printed near the set code in the bottom-left corner. This is the regulation mark. Cards rotating out of Standard format typically see a <strong>15–40% price drop</strong> on singles that were primarily purchased for competitive play. Reading this mark on a raw card tells you immediately whether you're looking at a Standard-legal card or a rotation candidate. It's a pricing signal hiding in a corner most people ignore.</p>
